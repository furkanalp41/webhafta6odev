# Mini Kitaplık Ödevi

Bu proje, React kullanılarak geliştirilmiş bir siber güvenlik temalı mini kitaplık uygulamasıdır.

## Özellikler
- Kitapları başlığa göre arama
- Kategoriye göre filtreleme
- Favorilere ekleme/çıkarma
- Arama geçmişini gösterme
